## 🧾 Sample Bug Report Format

```text
Title: [Module][Feature] Short description of the bug

Environment:
- Device / Browser:
- Build / Branch:
- Account / Role:

Pre-condition:
- User is logged in / User is on specific screen / Test data is prepared

Steps to Reproduce:
1. ...
2. ...
3. ...

Expected Result:
- Describe the correct behavior

Actual Result:
- Describe what actually happened

Severity:
- Critical / High / Medium / Low

Priority:
- High / Medium / Low

Evidence:
- Screenshot / video / logs / API response
```
